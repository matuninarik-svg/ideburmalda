Что такое Шаррокин?
Современные инструменты для написания кода на основе искусственного интеллекта забывают все, как только вы закрываете вкладку. Sharrowkin решает эту проблему.

Это настольная интегрированная среда разработки с поддержкой искусственного интеллекта с 4-уровневой системой постоянной памяти, которая изучает вашу кодовую базу, запоминает архитектурные решения и сохраняет контекст между сеансами работы и перезапусками, не отправляя ваш код в облако.

Характеристики
Постоянная 4-уровневая память
DSM — семантический векторный поиск решений в вашей кодовой базе
RLD — изучает успешные шаблоны и автоматически их использует
MemoryField — геббовское закрепление архитектурных концепций
TraceMemory — история выполнения для самокоррекции и предотвращения зацикливания
Параллельные субагенты
Запускайте несколько специализированных агентов, которые будут работать одновременно. Один пишет код, другой запускает тесты, третий проверяет. Каждый агент отображается в чате в виде интерактивной карточки — нажмите, чтобы посмотреть.

Контрольная точка и возврат
Перед каждым редактированием файла агент делает снимок исходного состояния. В чате появляется постоянная панель: вы можете принять изменения для каждого файла по отдельности или отменить все изменения одним щелчком мыши.

Встроенное Автозаполнение
По мере ввода текста появляются подсказки. Для принятия нажмите Tab, для отклонения — Esc. Напишите комментарий, например // build a burger menu и нажмите Tab, чтобы сгенерировать полную реализацию.

График проекта
Визуальный направленный граф всех зависимостей файлов. Фильтруйте по языку, нажимайте на узлы, чтобы просмотреть связи.

Беги

# Серверная часть
pip install -r requirements.txt
cp .env.example .env # добавьте свой ключ API
python main.py

# Фронтенд (новый терминал)
cd ui
npm install
npm run dev
Откройте http://localhost:5173 — настройте ключ LLM на экране регистрации.

Настольное приложение (Tauri)
cd ui
npm run tauri build

Архитектура
sharrowkin/ide/
├── main.py                 # FastAPI entrypoint
├── src/
│   ├── agent/
│   │   ├── core.py         # 5-phase agent loop
│   │   ├── tool_loop.py    # ReAct tool execution
│   │   └── agent_tools.py  # Tool implementations + sub-agents
│   ├── memory/
│   │   ├── dsm/            # Dynamic Segmented Memory
│   │   ├── rld/            # Recursive Latent DNA
│   │   ├── field.py        # MemoryField (Hebbian)
│   │   └── trace.py        # TraceMemory
│   ├── core/llm/           # LLM client (OpenAI/Anthropic/Gemini)
│   └── api/routers/        # REST + WebSocket endpoints
└── ui/                     # React + Vite + TypeScript
    └── src/
        ├── components/     # Editor, Chat, Terminal, Git, Graph
        ├── pages/          # Welcome, Settings, Onboarding
        └── state/          # Global IDE state
